### Nvidia GTC 2026: Jensen Huang Sees $1 Trillion in Orders Through 2027

- **URL:** https://www.cnbc.com/2026/03/16/nvidia-gtc-2026-ceo-jensen-huang-keynote-blackwell-vera-rubin.html

### Mistral Launches Leanstral: Open-Source Formal Proof Engineering Agent
- **Source:** Mistral AI
- **URL:** https://mistral.ai/news/leanstral
- **Summary:** Mistral released Leanstral, the first open-source code agent designed for Lean 4 formal proof engineering. With just 6B active parameters, it outperforms much larger models (Qwen3.5 397B, Kimi-K2.5 1T) and matches Claude Sonnet at 1/15th the cost. Released under Apache 2.0, Leanstral represents a significant step toward AI agents that can formally verify their own code — a potential breakthrough for safety-critical software development.

## Cybersecurity

### GlassWorm Campaign Compromises Hundreds of Python Repos via Stolen GitHub Tokens
- **Source:** The Hacker News, StepSecurity
- **URL:** https://thehackernews.com/2026/03/glassworm-attack-uses-stolen-github.html
- **Summary:** The GlassWorm malware campaign has escalated with a new "ForceMemo" variant that uses stolen GitHub tokens (harvested via malicious VS Code and Cursor extensions) to inject obfuscated malware into hundreds of Python repositories. The attack targets setup.py, main.py, and app.py files via force-pushes while preserving original commit metadata. The malware skips execution on Russian-locale systems.

### Google Patches 2 Actively Exploited Chrome Zero-Days
- **Source:** The Hacker News
- **URL:** https://thehackernews.com/2026/03/google-fixes-two-chrome-zero-days.html
- **Summary:** Google released emergency patches for two high-severity Chrome vulnerabilities being exploited in the wild: CVE-2026-3909 (out-of-bounds write in Skia 2D graphics library) and CVE-2026-3910 (inappropriate implementation in V8 engine). Both can lead to code execution. Fixed in Chrome 146.0.7680.75/76.

### Authorities Disrupt SocksEscort Proxy Botnet
- **Source:** The Hacker News
- **URL:** https://thehackernews.com/2026/03/authorities-disrupt-socksescort-proxy.html
- **Summary:** An international law enforcement operation dismantled SocksEscort, a criminal proxy service powered by the AVrecon malware. 4. **Robotics Emerges as VC Mega-Theme:** Mind Robotics ($500M), Rhoda AI ($450M), and related deals signal that robotics is joining AI infrastructure as a primary capital magnet, with industrial automation at the forefront.

6. **Fed Decision Looms:** Wednesday's rate decision comes amid an exceptionally complex backdrop — Iran-driven oil inflation vs. weakening labor market. The outcome will set the tone for the rest of March.
